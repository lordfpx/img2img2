# MiniPix Converter

MiniPix is a lightweight React + Vite + Tailwind web app that converts batches of images in the browser to PNG, JPEG, WebP, and GIF.

## Quick start

```bash
npm install
npm run dev
```

Open `http://localhost:5173` to access the UI.

## Build & preview

```bash
npm run build
npm run preview
```
